# Odin Recipes

### 📝 Description
This is the first project from **The Odin Project** Foundations curriculum. The goal was to create a basic recipe website using only HTML to demonstrate a solid understanding of structural web development before moving on to CSS and design.

### 🛠️ Skills Demonstrated
Through this project, I have showcased proficiency in:
* **HTML Structure:** Utilizing proper document hierarchy (`<html>`, `<head>`, `<body>`).
* **Text Formatting:** Working with headings (`h1`-`h3`) and paragraphs.
* **Lists:** Implementing both ordered (`<ol>`) and unordered (`<ul>`) lists for ingredients and steps.
* **Media & Links:** Embedding images and creating functional internal/external hyperlinks.
* **Git/GitHub:** Managing version control and deploying a live project via GitHub Pages.
